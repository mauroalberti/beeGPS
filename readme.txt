Track GPS location
------------------

Plugin for QGIS to display a gps coordinate in "real time"
Read gps data from gpsd http://gpsd.berlios.de/

Code for CanvasMarker.py from qgis-mapper
   http://urtax.ms.mff.cuni.cz/qgis-mapper/trac/
